
BitKultur • GitHub Pages csomag (HTML + reszponzív táblák)

LÉPÉSEK
1) Új GitHub repó:
   - Név: username.github.io  (username = GitHub felhasználóneved) • Publikus

2) Fájlok feltöltése:
   - index.html
   - /assets/css/site.css
   - /assets/img/ (PNG képek)

3) GitHub Pages bekapcsolása:
   - Settings → Pages → Source: Deploy from a branch
   - Branch: main (vagy master), Folder: /(root)
   - Mentés → 1–2 perc múlva él az oldal: https://username.github.io

4) Medium integráció:
   - Táblázatokat Mediumon képként (screenshot) használd, vagy linkeld a GitHub Pages oldalt.
   - Medium cikkben a More settings alatt add meg a Canonical linket a Pages URL-re.

5) Extra:
   - Ha több cikket szeretnél, hozz létre aloldalakat (pl. pages/opreturn.html) és linkeld őket a nyitóoldalról.
